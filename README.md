# AI Governance & Cybersecurity

Course repository for AI governance with hands-on cybersecurity. 

## Quick start

1. Clone the git project 
`git clone [project repo]`

2. Install all required dependencies using 
`pip install -r requirmenent.txt` 

3. Start the MCP server using `python gemini_mcp.py`

4. In a new terminal, start the app using `python ui_app.py`

Note: We are using a strict set of credentials which is available to use for everyone. No logins will be required.
